Thank you for downloading the FC_AEMCraft plugin created by Destro168.

Bukkit Dev Page -> http://dev.bukkit.org/server-mods/fc_aemcraft/
FC_AEMCraft Changelog & Source -> https://github.com/Destro168/FC_AEMCraft/blob/master/Docs/Changelog

Description:
FC_AEMCraft is a custom plugin designed for the AEMCraft server. There are two major features to this plugin. The first is the player tracking system and the second is the material harvesting for cash system.

The main command for this plugin is /aem.
